Farcaster Auditor (Terminal UI) - Fixed

How to run:

1. Install Node deps (from this folder):

   npm install

2. Put your Neynar API key in `.env`:

   NEYNAR_API_KEY=YOUR_KEY_HERE
   PORT=3000

3. Start backend + static server:

   npm start

4. Open browser at:

   http://localhost:3000

   In the input, type:
   - Farcaster username (e.g. dwr, v, something.base.eth)
   - OR numeric FID (e.g. 1407742)

   Then press EXECUTE.
